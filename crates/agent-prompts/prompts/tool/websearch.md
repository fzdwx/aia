Search the web using Exa AI. Performs real-time web searches and can scrape content from specific URLs.

Usage:
- Provides up-to-date information for current events and recent data.
- Supports configurable result counts and returns content from the most relevant websites.
- Use this tool when you need information beyond the model knowledge cutoff.
- Searches are performed automatically within a single API call.
- Supports live crawling modes: `fallback` or `preferred`.
- Supports search types: `auto`, `fast`, or `deep`.
- `contextMaxCharacters` controls how much result context is returned for LLM consumption.
- Domain filtering and advanced search options may be available upstream through Exa search semantics.

The current year is `{{year}}`. You MUST use this year when searching for recent information or current events.
- Example: if the current year is `2026` and the user asks for latest AI news, search for `AI news 2026`, not `AI news 2025`.
