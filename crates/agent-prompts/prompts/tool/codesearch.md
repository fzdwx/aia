Search and get relevant context for any programming task using Exa Code API.

Usage:
- Provides high-quality and fresh context for libraries, SDKs, APIs, and programming concepts.
- Use this tool for programming questions when you need documentation, examples, or API references beyond the local workspace.
- Returns comprehensive code examples, documentation snippets, and implementation references.
- The `tokensNum` parameter accepts `1000` to `50000`; default `5000` works well for most tasks.
- Use lower `tokensNum` values for focused lookups and higher values for broader documentation context.
- Example queries: `React useState hook examples`, `Python pandas dataframe filtering`, `Express.js middleware`.
