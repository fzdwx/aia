Render an interactive HTML/SVG visualization in the chat. IMPORTANT: Call widgetReadme once before your first widgetRenderer call to load design guidelines. Use this tool when the user asks you to visualize, diagram, illustrate, or explain something visually. Great for: algorithm visualizations, architecture diagrams, flowcharts, interactive simulations, math plots, data dashboards, step-by-step animations, and any visual explanation. The HTML runs in a sandboxed iframe with theming support. Always prefer this over describing visuals in text.
